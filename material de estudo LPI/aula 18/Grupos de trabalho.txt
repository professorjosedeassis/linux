Grupos são usados para gerenciamento de usuários que tenham algo em comum.

groups ->exibe os grupos que o usuário pertence

/etc/group -> arquivo de configuração dos grupos
/etc/gshadow -> arquivo de senhas de gtupos

groupadd nome_grupo -> cria um grupo de trabalho
gpasswd -a usuario nome_grupo -> adiciona o usuário ao grupo
gpasswd -d usuario nome_grupo -> remove o usuário do grupo
groupmod -n novo_nome_grupo nome_grupo -> renomeia o nome do grupo

id -> exibe os ids do usuário e grupo

Exemplo:
id root

Observação: Usar o vigr para edição segura do arquivo /etc/group
            e  vigr -s -> ediçaõ segura do /etc/gshadow

getent group usuario -> mostra informações sobre os grupos que o usuário pertence
