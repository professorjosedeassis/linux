Instalar o Linux (Debian)

tty -> terminais

<alt><F1>---<F6> (mudar os terminais)
<alt><f7> (voltar para interface gráfica)

~ usuário logado no seu diretório padrão

# -> usuário administrador
$ -> usuário comum

___________________________________________________
Teclas de atalho:
<ctrl D> -> deslogar
setas (cima, baixo) -> repetem os últimos comandos
<tab> -> completa um comando ou sentença
q ou <ctrl><c> -> sair, parar, etc

___________________________________________________
Cores e significados:

azul escuro -> diretório
azul claro -> atalho (link)
verde -> arquivos executáveis
amarelo -> arquivos de driver(hardware)
rosa -> arquivo de imagem
vermelho -> pacotes (.deb .rpm)
branco -> arquivo de texto comum

__________________________________________________
Diretórios e significados:

/ -> raiz
/root -> diretório do administrador
/home -> diretório de usuários
/bin -> executáveis do usuário
/sbin -> executáveis do root
/var -> logs e principais serviços
/dev -> arquivos de drivers
/etc -> arquivos de configuração
/boot -> arquivos de inicialização
/tmp -> arquivos temporários
swap -> memória virtual


Comandos básicos:
exit ou logout -> deslogar
pwd -> verifica o diretório atual
clear -> limpar a tela
date -> exibir data e hora
cal -> exibir um calendário
shutdown -h now -> desligar o servidor
history -> exibe os últimos 1000 comandos digitados

*** No linux tudo é arquivo ou diretório
*** Não existe extensão (.txt .docx .exe) no Linux

cd -> comando para ir a um diretório
cd .. -> voltar um nível de diretório
cd <enter> -> vai para o diretório padrão do usuário
ls -> listar arquivos e diretórios
ls | less -> | (canalizador) less -> exibir aos poucos
