/etc/passwd -> principal arquivo de configuração de usuários

### Anatomia do passwd

joseassis:x:1000:1000:José de Assis,,,:/home/joseassis:/bin/bash

: -> delimitador dos campos ("tabela")

joseassis -> nome do usuário
x -> referência a senha
1000 -> id do usuário
1000 -> id do grupo do usuário
GECOS -> informações do usuário (nome, fone, etc)
/home/joseassis -> diretório padrão do usuário
/bin/bash -> executar o shell padrão

### ferramentas para filtrar este arquivo

cut -d ':' -f 1,5 /etc/passwd (exibe as colunas 1 e 5 do aequivo /etc/passwd)

cut -> "cat" (exibir "pedaços" do rexto")
-d -> delimitador
-f -> field(coluna) 

### Para edição do arquivo /etc/passwd usar o vipw

(editor vi em modo seguro otimizado para a edição do arquivo /etc/passwd)

___________________________________________________________________________________

passwd -> muda a senha do root
passwd usuário -> muda a senha do usuário

adduser teste -> adiciona um novo usuário no linux usando um "template" pré definido
/etc/skel -> template de criação de usuários (scripts)
chfn -> altera informações de GECOS

useradd teste2 -> adiciona um novo usuário sem usar o template

useradd -m teste3 -> adiciona um novo usuário sem usar o template, porém cria o diretório padrão

useradd -d /var/public_html teste4 -> adiciona um novo usuário sem usar o template, criando um diretório padrão em ouro caminho.

userdel -> remove o usuário
userdel -r -> remove o usuário e seu diretório padrão

__________________________________________________________________
/etc/shadow -> arquivo de armazenamento das senhas dos usuários 

vipw -s -> edição segura do arquivo shadow
